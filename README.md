## Course Outcomes

After completing this course, students will be able to:

1. Explain the fundamental concepts and challenges of Natural Language Processing.
2. Apply text preprocessing and statistical techniques to natural language data.
3. Analyze lexical, syntactic, semantic, and pragmatic aspects of natural language.
4. Design NLP solutions for applications such as text classification, question answering, machine translation, and conversational agents.

## Course Contents

### Unit 1 – Introduction

* Importance of NLP
* Language Models
* NLP Processing Pipeline
* Applications of NLP
* Challenges and Ambiguity in Natural Language

### Unit 2 – Text Pre-processing

* Character Encoding
* Language and Corpus Dependence
* Tokenization
* Sentence Segmentation

### Unit 3 – Lexical and Syntactic Analysis

* Morphological Analysis
* Finite-State Morphology
* Part-of-Speech Tagging
* Parsing Techniques
* Constraint-Based Grammars

### Unit 4 – Semantic Analysis

* Natural Language Semantics

* Lexical Semantics

### Unit 5 – Empirical and Statistical Approaches

* Corpus Creation
* Treebank Annotation
* Statistical Parsing
* Hidden Markov Models
* Viterbi Algorithm
* Word Similarity
* Word Sense Disambiguation

### Unit 6 – Pragmatics

* Discourse
* Dialogue Systems
* Conversational Agents

* Reference Resolution
* Pronoun Resolution

### Unit 7 – Applications

* Chatbots
* Machine Translation
* Spam Filtering
* NLP-Based Search Engines
